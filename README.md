# trading-model

Historical data, research, parameter modeling, and backtest-focused codebase.

This repository should retain:
- historical data ingestion for research use
- dataset construction
- market-state / regime research
- parameter utility modeling
- backtests and offline evaluation
- research reports and model artifacts

This repository should not remain the home for realtime trading / live execution code.
Those responsibilities are moving to:
- <https://github.com/gilmore307/quantitative-trading>
